# Frontend (React + TypeScript + Vite)

## Requisitos

- Node.js 20+

## Arranque

```bash
npm install
npm run dev
```

App en `http://localhost:5173`

## Flujo API usado

- `GET /api/entries`
- `POST /api/entries`
- `POST /api/entries/{entryId}/resources/text`
- `POST /api/entries/{entryId}/resources/link`
- `POST /api/entries/{entryId}/resources/media`

## Build

```bash
npm run build
npm run preview
```
